# IoT Smart Energy Monitoring System

## Description
This project monitors real-time electricity consumption using IoT technology. It measures voltage and current, calculates power usage, and sends data to the cloud for remote monitoring.

## Features
- Real-time monitoring
- Cloud integration
- Energy usage analysis

## Technologies Used
- Arduino
- Sensors
- IoT Platform
